## 🚀 Scripts Overview  

| Script | Purpose | Key Use Case |
|--------|---------|--------------|
| 🔹 **backup.sh** | Automates backup of files/directories by creating compressed archives | Data protection & scheduled backups |
| 🔹 **log_analyzer.sh** | Analyzes log files to extract insights (error patterns, keywords) | Debugging & monitoring system/application logs |
| 🔹 **log_rotate.sh** | Implements log rotation, archives old logs, manages log size | Prevents excessive disk space usage |
| 🔹 **maintenance.sh** | Performs routine system maintenance (cleanup, optimization) | Keeps system healthy & efficient |
| 🔹 **sample_logs_generator.sh** | Generates sample log files for testing | Simulates real-world scenarios for analysis/rotation |
| **shell_scripting_cheatsheet.md** | Quick reference guide for essential shell scripting concepts & commands | Handy learning & interview prep resource |
